# Secure Online Fee Payment System

## Description
A secure web-based application designed to demonstrate core cyber security concepts such as authentication, authorization, encryption, hashing, digital signatures, and encoding.

## Security Features
- Single-factor and Multi-factor Authentication (OTP-based)
- Role-Based Access Control (Admin, Student, Accountant)
- AES Encryption for sensitive data
- Password hashing with salt
- Digital signatures using hash
- Base64 encoding

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express
- Database: MongoDB
